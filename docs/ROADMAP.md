## Investor Narrative

1. **Platform, not apps** — vertical domains scale; topics are pages
2. **SEO moat** — knowledge graph + programmatic SEO at scale
3. **Self-build** — AI generates topics into registry, not new codebases
4. **Transparency** — public build log proves execution velocity
5. **Unit economics** — $4 / $18 tiers per vertical, shared infra
