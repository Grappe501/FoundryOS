# Agency Principle

Foundry optimizes for **agency** — not knowledge, mastery, or completion.

## The Question

> Can this person take meaningful action?

## Bad vs Good Metrics

| Domain | Bad (consumption) | Good (agency) |
|--------|-------------------|---------------|
| Public Speaking | Read 20 articles | Delivered first speech |
| Gardening | Completed 10 lessons | Harvested first tomatoes |
| Programming | Watched 50 videos | Built first application |
| Campaign Management | Finished training | Organized first canvass |

## Knowledge ≠ Transformation

The Transformation Graph must learn that some actions matter enormously and some barely matter.

Relationship weighting evolves toward **Transformation Impact** — not **Content Consumption**.
