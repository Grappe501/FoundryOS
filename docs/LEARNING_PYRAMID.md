# Foundry Learning Pyramid

Five layers on every academic domain.

| Layer | Question | Examples |
|-------|----------|----------|
| 1 Definitions | What is it? | Derivative, API, Primary source |
| 2 Concepts | Why does it work? | Conservation of energy, Abstraction |
| 3 Execution | How do I use it? | Problems, labs, integrations |
| 4 Projects | Build something | CRM, water analysis, oral history |
| 5 Mastery | Teach it | Mentor, guides, study groups |

---

## Road to AI Builder

**Definitions:** API, Database, Function, Prompt

**Concepts:** Abstraction, architecture, systems

**Execution:** Calculator, todo app, API integration

**Projects:** CRM, Campaign OS, Foundry module

**Mastery:** Tutorials, mentor builders, templates
