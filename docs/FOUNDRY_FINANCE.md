# Foundry Finance — Financial Independence Domain

> Not Stocks. Not Accounting. **Financial Independence.**

---

## The Mistake to Avoid

Do **not** launch:

```txt
Stocks app
Accounting course
```

People don't wake up saying *"I want accounting."*

They say:

```txt
I want to understand money.
I want to start a business.
I want financial freedom.
```

Launch as **Business & Money** framing. Accounting is a **path**, not a product.
