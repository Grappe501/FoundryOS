# Reflection Engine — PASS-010

Foundry knows what you want, what you know, and what you did.

Without reflection, it does not know **what you learned**.

That sounds small. It is enormous.

## Human Growth Loop

Most platforms stop here:

```txt
Learn → Do
```

Foundry becomes:

```txt
Learn → Do → Reflect → Adjust → Grow
```

## Every Project Ends With

```txt
What happened?
What did you learn?
What should happen next?
```

## Exemplar Reflections

### Public Speaking — Deliver first speech

- What surprised you?
- What went well?
- What would you change?
- What did you learn?

### Gardening — Grow first tomatoes

- What worked?
- What failed?
- What would you try differently?

### AI Builder — Build first application

- What was hardest?
- What skill was missing?
- What helped most?

## Transformation Intelligence Loop

```txt
Goal → Path → Project → Action → Evidence → Reflection → Insight → Next Best Action
```

This loop repeats forever.

## The Real Gold

Projects completed are signals. **Lessons learned** are wisdom.

The moat: millions of people reflecting on transformation. Nobody owns that.
