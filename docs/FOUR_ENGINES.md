# Four Engines

Every vertical — Poker, Bourbon, Public Speaking, Magic — uses the same architecture.

| Engine | Question | Package |
|--------|----------|---------|
| **Knowledge** | What is it? | `@foundry/encyclopedia-engine` |
| **Mastery** | How do I become good at it? | `@foundry/path-engine` |
| **Project** | What do I actually do? | `@foundry/project-engine` |
| **Community** | Who do I do it with? | `@foundry/community-engine` |

Plus: **Identity** + **Legacy**

---

## Self-Assembly Stack

```txt
Knowledge + Mastery + Projects + Community + Identity + Legacy
```

For **any subject a human cares about.**
