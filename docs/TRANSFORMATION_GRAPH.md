# Transformation Graph — Foundry's Moat

PASS-009 is the first pass about **defensibility**, not features.

## The Real Moat: Transformation Intelligence

Not content. Not AI. Not courses. Not communities. Not knowledge alone. Not projects alone.

```txt
We know what actions help people grow.
```

The dataset: **millions of people reflecting on transformation.** Nobody owns that.

## Human Development Loop

```txt
Potential → Outcome → Path → Project → Action → Evidence → Reflection → Insight → Next Best Action → Contribution → Mentorship → Legacy
```

That is no longer a learning platform. That is a **human development loop**.
