# Foundry Life Graph

Five connected graphs. Everything connects.

```txt
knowledge_graph
ownership_graph
identity_graph
project_graph
community_graph
```

---

## Example Journey

```txt
Steve
  → Road to Bourbon Master
  → Completed Project: Blind Tasting Night
  → Hosted Bourbon Tasting
  → Created Club
  → Mentored Members
  → Earned Bourbon Steward
```

This story is more valuable than any AI-generated article.

---

## Graph Roles

| Graph | Powers |
|-------|--------|
| Knowledge | Encyclopedia, entities, relationships |
| Ownership | Collections, reviews, rankings, shelf |
| Identity | Mastery titles, portable profile |
| Project | Knowledge → action → visible expertise |
| Community | Clubs, challenges, mentorship |

---

## Journey Memory

`journey_events` — Foundry remembers journeys over years.

Event types: started, milestone, project, club, mentor, mastery
