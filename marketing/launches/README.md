# Launch Factory Output

Each domain launched via `npm run launch:domain <slug>` gets a folder here with:

- `launch.json` — manifest
- `LAUNCH_BRIEF.md`
- `LAUNCH_CHECKLIST.md`
- `content-calendar-30d.md`
- `email-sequence.md`
- `social-calendar.md`
- `seo-cluster-map.json`
- `growth-plan.json`
- `monetization-plan.md`

Registry: `registry.json` — tracks Domain Launch Velocity.

```bash
npm run launch:domain -- ai-builder
```

Pass gate: **Users · Revenue · Retention** — not another system.
