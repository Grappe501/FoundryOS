# Acquisition Events

Temporary traffic spikes that feed **permanent domains** — not standalone products.

## Growth Factory Funnel

```txt
Traffic Event → Permanent Domain → Transformation System → Retention → Revenue
```

## Examples

| Event | Permanent Domain |
|-------|------------------|
| World Cup 2026 | Soccer |
| Midterm Elections 2026 | Civic Engagement |
| Gardening Season | Master Gardener |
| Bourbon Heritage Month | Bourbon |
| AI Boom | AI Builder |

Event briefs live in sibling folders (e.g. `world-cup-2026/`, `midterm-elections-2026/`).

Scored in `registry.json` using `scoring-rubric.json`.
